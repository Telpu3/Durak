# 🃏 DurakGame
**Приложение «Дурак»** — классическая карточная игра для двух игроков (человек против компьютера), разработанная с использованием объектно-ориентированного программирования и архитектуры Windows Forms.

**Цель игры:** Избавиться от всех карт в руке раньше соперника. Когда колода пуста, а у одного из игроков не осталось карт — он объявляется победителем.

---

## 🎮 Основные возможности

### Игровая механика
- **Колода:** 36 карт (от 6 до туза четырёх мастей)
- **Козырь:** Определяется случайно в начале партии (нижняя карта колоды)
- **Первый ход:** Игрок с младшим козырем. При отсутствии козырей — случайный выбор
- **Атака:** Атакующий выкладывает карту на стол. Можно подкидывать карты того же достоинства, что уже лежат на столе
- **Защита:** Защитник должен побить карту картой той же масти большего достоинства или козырем
- **Добор:** После каждого раунда игроки добирают карты из колоды до 6 (сначала атакующий, затем защитник)
- **Смена ролей:** После успешного отбоя роли меняются. При взятии карт — остаются прежними

### Искусственный интеллект компьютера
- **При атаке:** Выбирает наименьшую некозырную карту (если стол пуст) или подкидывает карты подходящего ранга, стараясь не тратить козыри
- **При защите:** Среди подходящих карт выбирает наименьшую некозырную (приоритет) либо наименьшую козырную
- **Асинхронные паузы:** Имитация размышления (0.4 сек) и пауза для просмотра результата (1.8 сек)

### Условия победы
- У игрока 0 карт в руке **И** колода пуста
- Досрочное завершение по нажатию кнопки «Новая игра»

### Обработка ошибок
- Двухуровневая система валидации (форма + библиотека классов)
- Информативные сообщения при недопустимых действиях
- Восстановление отображения стола при некорректной защите

## 🚀 Как запустить

### Через Visual Studio
1. Откройте файл `Durak.sln`
2. Убедитесь, что стартовым проектом установлен `K`
3. Нажмите **F5** для запуска с отладкой или **Ctrl+F5** без отладки
4. Нажмите **«Новая игра»** для начала партии
---
### Особенности программы
1. Трёхуровневая архитектура: чёткое разделение на библиотеку классов (игровая логика), пользовательский интерфейс (Windows Forms) и модульные тесты.
2. Умный искусственный интеллект: компьютер старается экономить козыри и сильные карты, выбирая наименьшую подходящую карту для атаки и защиты.
3. Паузы: имитация размышления перед ходом и пауза после действия, чтобы игрок успевал следить за происходящим.
4. Двухуровневая обработка ошибок: валидация на уровне формы и на уровне бизнес-логики с информативными сообщениями.
5. Модульное тестирование: 25 автоматических тестов, покрывающих все ключевые классы и методы.
